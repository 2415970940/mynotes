vim

命令模式（编辑模式）默认
输入模式
末行模式
	set nu  行号
	100d  删除第100行

模式转换
	命令模式到输入模式
	i  在当前光标前转为输入模式
	a  在当前光标后转为输入模式
	o  在当前光标的下一行

	I  在当前光标行首
	A  在当前光标行尾
	O  在当前光标的上一行

	输入模式到命令模式
	esc

	编辑模式到末行模式
	：


打开文件
	vim + file 光标在文本尾部
	vim +/pattern file 第一次匹配的行号

关闭文件
	末行模式
	：wq  保存退出
	:q!  强行退出
	:wq == :x
	命令模式
	ZZ（大写）
编辑模式
		移动光标（编辑模式）
			hjkl：左下上右
			#h，移动#个字符
			
		单词移动
			w 移动下一个单词词首
			e 调至词尾
			b 词首

		行内跳转
			0：绝对行首
			^: 行首（非空白符）
			$: 行尾

		行间跳转
			G:最后一行
			#G
			末行模式 :#回车

		翻屏
			ctrl+f 向下
			ctrl+b 向上

			ctrl+d 向下半屏
			ctrl+u 向上半屏
			
		删除单个字符
			x
			#x
		删除命令
			d
			和跳转命令组合
			dw删除单词
			dd删除一行
			2dd，删除两行
			末行模式 :.,100d 当前行到100行
					 :.,$-2  当前行到倒数第2行
		
		最后一次删除的粘贴 
			p 下方或者后方
			P  上方或前方
		复制
			y
		先删除内容，再转换输入模式
			c
		替换单个字符
			r
			R替换模式
		撤销
			u
		撤销撤销
			ctrl+r
		重复前一次操作
			.

		v
